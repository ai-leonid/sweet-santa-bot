# Контекст приложения "Sweet Santa Bot"

"Sweet Santa Bot" — это Telegram Mini App (TMA) для организации и проведения игры "Тайный Санта".
Приложение позволяет пользователям создавать "Игры" (комнаты для жеребьевки) и приглашать в них друзей через шеринг ссылки.
Ключевая особенность — система "стоп-листов", позволяющая каждому участнику указать людей, которым он не должен дарить подарок (с опцией автоматического взаимного исключения).
Создатель игры имеет расширенные права: он может добавлять "оффлайн-игроков" (без Telegram-аккаунта) и запускать процесс жеребьевки.
После старта игры каждый участник заходит в приложение, чтобы узнать, кому он дарит подарок; имя подопечного скрыто под спойлером.
Результаты оффлайн-игроков доступны для просмотра на устройстве создателя игры в отдельном интерфейсе.
Технически проект реализован как монолит на Next.js (App Router) с использованием Server Actions для бэкенд-логики.
В качестве базы данных используется PostgreSQL, взаимодействие с ней происходит через Prisma ORM.
UI построен на библиотеке компонентов shadcn/ui с стилизацией через Tailwind CSS.
Авторизация пользователей осуществляется через данные инициализации Telegram WebApp (initData).
